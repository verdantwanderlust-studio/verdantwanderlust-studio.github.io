邊界

什麼時候找我

AI 助理是有界線的。下面這些事它不會做,你會看到它跟你說「這要找 Jason」。

AI 幫你
  • 改任何頁面的文字
  • 換、加、刪照片
  • 調整顏色、字級、間距數字
  • 加新案例、新服務項目
  • 整理圖片資料夾
找 Jason
  • 改網站的版面結構(layout)
  • 加新功能(表單、訂閱、付款)
  • 網站打不開、出現紅色錯誤
  • 整體改版(重新設計)
  • 任何讓你緊張的請求

不確定「這個算不算 AI 能做的」就直接問它。它會老實跟你說「這要找 Jason」,不會亂改。

07 / 11
安全機制

為什麼不會弄壞

01

先確認、後動手

AI 動任何東西前一定會說「我要做 X,你 OK 嗎?」。你不點頭,它什麼都不會改。

02

先預覽、後上線

改完只給你「preview 網址」,只有你看得到。你說「發布」,客人才看得到新版。

03

隨時救得回來

每次改動都有歷史紀錄。發布後不喜歡,跟 AI 說「上一版」就還原了。

0
需要你記的指令
3 步
確認 → 預覽 → 發布
∞
可以還原的次數

常用句型

你可以這樣講

不用記、不用學。這只是讓你看一下「原來這麼簡單」。

你說 AI 會做
「首頁標題改成 OOO」 改文字 → 給你 preview → 等你說發布
「我傳一張新照片,放到作品集第一張」 優化圖片 → 替換 → preview
「這頁的字看起來太小」 問你大概要多大,然後調 → preview
「加一個新案例,叫 OOO,在 OO 路」 問你要哪些照片、要寫什麼,然後生成新頁面 → preview
「上一版」 / 「還原」 退回前一個版本
「網站還好嗎?」 檢查狀態、用量、最近改了什麼
「發布」 / 「上線」 把 preview 推到正式網站

今天

我們一起弄好這 5 件事

只有「要用你的信箱、手機、信用卡」的事需要你本人。其他都 Jason 回家弄。

01

訂閱 AI 工具

ChatGPT Plus 或 Claude Pro,擇一。月費 $20。用你的信箱 + 你的信用卡。

5 分鐘
02

開 GitHub 帳號

放網站程式的地方。用你的信箱 + 手機 2FA。開好把 Jason 加進去當協作者。

3 分鐘
03

開 Cloudflare 帳號

網站「住」的地方。用你的信箱 + 手機 2FA。開好把 Jason 加進去當 member。

3 分鐘
04

決定網址

想用什麼網域名(.com / .tw / .design...)。想好就直接買,大約 $10-15 / 年。

5 分鐘
05

確認手機 2FA

能收簡訊就好,或裝 Google Authenticator / 1Password。前面註冊都會用到。

1 分鐘

所有帳號都是你的 · Jason 只是被你授權「代你操作」。
之後如果想自己管,隨時可以把他踢掉。
